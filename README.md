# 🎓 Forever In Our Hearts — Batch of 2026

![Farewell Banner](https://images.unsplash.com/photo-1523050335456-adabbf72c766?w=1200&q=80)

> "Not goodbye — just until we meet again."

A premium, cinematic farewell experience built to preserve the laughter, the lessons, and the lifelong bonds of the Batch of 2026. This isn't just a website; it's a digital time capsule.

[**✨ View Live Experience**](https://ux-dice.github.io/farewell-2026/)

---

## 🎬 The Experience

This project is designed with a **Cinematic First** approach, utilizing modern web technologies to create an emotional journey through time.

### 🌟 Key Features
- **🎞️ Cinematic Hero**: An immersive video introduction that sets the emotional tone.
- **📜 Memory Ticker**: A continuous flow of shared moments and inside jokes.
- **📸 Interactive Gallery**: A custom-built masonry grid with lightbox support to relive every snapshot.
- **🎶 Premium Music Player**: A floating, glassmorphism-styled player for that perfect nostalgic soundtrack.
- **🎭 Superlatives**: Celebrating the unique personalities that made the batch special.
- **💫 Fluid Animations**: Powered by GSAP and ScrollTrigger for a buttery-smooth, high-end feel.

## 🚀 Getting Started

To run this time capsule on your local machine:

1. **Clone the repository**
   ```bash
   git clone https://github.com/ux-dice/farewell-2026.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Launch the experience**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
